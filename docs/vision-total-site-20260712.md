# アートメイク比較メディア：総合サイトの理想像と中期実行計画（2026-07-12策定）

## 0. 現状診断（GSC直近28日）
- **0click / 319imp**。大半が **pos24〜80（page3以降）** ＝コンテンツはあるが順位が付いていない“立ち上がり期”。
- 需要はある：「dazzy skin clinic」155imp・pos26.6／技法クエリ（3D/4D/6Dストローク）／指名レビュー系。→ **上位化できれば刈れる**。
- 資産：compare21・review11・knowledge32・parts5・purpose9・area12（土台は厚い）。
- 差別化の芽：AMG/Gメディカル/除去特集で実証した「**公式実額の横断比較**」＝競合（アグリゲーター・個人ブログ）に無い一次情報。

## 1. 理想像（Vision）
> **「アートメイクを検討する人が、①どのクリニックが自分に合うか（料金・技法・実績・口コミ）②部位・目的・エリアで最適な選択 ③施術の不安（痛み・持ち・失敗・除去）を全て解消できる。医療広告を守った“信頼できるアートメイク比較の総合ナビ”」**

### 3本柱
1. **比較（意思決定支援）**：クリニック比較で「どっち／いくら」に即答。差別化＝**公式実額の横断比較**（競合が持たない一次情報）。
2. **網羅（総合性）**：クリニック×部位×目的×エリア×知識×技法の6軸を相互リンクで網羅。
3. **信頼（E-E-A-T・医療広告順守）**：公式一次情報・確認日・**個人差/医療行為の明記**・最上級禁止。一次情報検証で競合に優位。

## 2. 中期実行計画（新規サイト＝まず“ランクさせる”が主眼）

### P0【土台】ランク獲得のためのE-E-A-T・内部リンク・技術SEO
- 主要ページ（比較・レビュー）の**内部リンク網を厚く**し、クロール・評価を集中（現状は個別ページが孤立気味）。
- **E-E-A-Tシグナル**：全ページに監修/運営者情報・公式確認日・一次情報出典を明示（YMYL領域の必須要件）。
- **sitemap整備**（現状要確認）・構造化データ（Article/FAQ/BreadcrumbはOK、加えてクリニックのReview/AggregateRatingは自作評点禁止のため使わず、比較表の充実で対応）。

### P1【勝てるクエリに集中】指名・比較で上位化
- **指名レビュー**：「dazzy skin clinic」（155imp）「◯◯クリニック レビュー」等、既に需要のある指名クエリのレビューを強化しpage3→page1へ。
- **比較クエリ（競合空白）**：「◯◯ vs △△ どっち」を厚くする（AMG/Gメディカルで実証済みの型）。主要クリニックの1対1比較を計画的に増設。

### P2【網羅性】6軸の穴埋め
- **クリニック軸**：主要クリニックのレビュー拡充（公式実額・技法・口コミ傾向）。
- **部位軸**：眉/リップ/アイライン/ヘアライン/ほくろ 各の相場・技法・注意点を深掘り。
- **目的軸**：40-50代/ブライダル/メンズ/医療（脱毛症・がん）/モニター。
- **エリア軸**：主要都市のクリニック比較（銀座・名古屋・大阪…）。
- **技法軸**：3D/4D/6D/毛並み/パウダーの違い（技法クエリに実需要）。
- **知識軸**：痛み/持ち/失敗/除去/ダウンタイム（32ページ既存を相互リンクで束ねる）。

### P3【差別化】公式実額の横断比較を主役化
- 除去特集のような「**実額横断比較表**」を料金・技法・部位別に展開。
- /compare/price を主役化し、全レビュー・知識から送客。

### P4【医療広告×信頼運用】
- 最上級・効果保証・体験談誘引の禁止、**個人差/医療行為の明記**、症例写真使用時は費用・リスク併記。出典・確認日を全社徹底。
- 料金の定期再実査（鮮度）＝【年月】運用。

### P5【CV動線】
- 「不安/比較クエリ → レビュー/比較 → 無料カウンセリング予約」を全ページで貫通。

## 3. 測定と運用
- 立ち上がり期は**順位（pos）とインデックスの改善**を主指標に（clickはその後）。GSCでpage3→page1への浮上を週次で判定。
- 上位化したページの型を横展開（指名レビュー・1対1比較の勝ち型）。

## 4. 90日ロードマップ（目安）
- 〜2週：P0（内部リンク網・E-E-A-T・sitemap）→クロール改善
- 〜1ヶ月：P1（指名レビュー強化＋比較クエリ増設）→ page3→page1の初動
- 〜2ヶ月：P2（部位・技法・目的軸の深掘り）＋P3（実額横断比較の主役化）
- 〜3ヶ月：6軸の内部リンク網完成＋鮮度運用定着 → “総合ナビ”として評価が付き始める
